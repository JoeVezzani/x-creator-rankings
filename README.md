# X Creator Rankings

Track and compare X creator engagement rankings using the LunarCrush API. Follower count is a vanity metric. Engagement is what matters.

## Quick Start

```bash
git clone https://github.com/JoeVezzani/x-creator-rankings.git
cd x-creator-rankings
```

Grab an API key at [lunarcrush.com/developers](https://lunarcrush.com/developers).

```bash
# Look up any creator
LUNARCRUSH_API_KEY=your_key node index.js PopBase

# Compare creators head-to-head
LUNARCRUSH_API_KEY=your_key node compare.js PopBase Cristiano FabrizioRomano NASA
```
